# Where the AI Jobs Are  
**DS 2023 – Communicating with Data | Final Project**

This repository contains my final project for DS 2023, exploring how AI-related job growth varies across industries in terms of projected demand, pay, remote work flexibility, and automation risk.

## Project Overview
The goal of this project is to communicate an interesting pattern in the data through an infographic. Using industry-level summaries, the project highlights that AI job growth is unevenly distributed and that high opportunity often comes with important tradeoffs.

## Contents
- **Notebook 1 – Data Setup**  
  Documents data acquisition, source background, and feature descriptions.

- **Notebook 2 – Exploration**  
  Contains exploratory visualizations and data transformations used to identify key patterns.

- **Notebook 3 – Final Product**  
  Presents the final infographic, project narrative, and complete project manifest.

- **Final Infographic (PNG & SVG)**  
  Poster-style visualization designed to communicate findings to a general audience.

- **Storyboard Images**  
  Early sketches used to plan the infographic layout and narrative.
